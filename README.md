# QuickWash – Online Laundry Service Platform 🧺

QuickWash is a web-based laundry management platform developed using Java Servlets, JSP, JDBC, HTML, CSS, and MySQL. The system simplifies laundry booking, order tracking, and service management through a responsive and user-friendly interface.

## 🚀 Features

* User registration and secure login system
* Online laundry service booking
* Order status tracking
* Admin panel for managing orders and customers
* Service and pricing management
* Responsive UI for better user experience
* Database-driven order management system

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Java Servlets, JSP
* Database: MySQL
* Connectivity: JDBC
* Server: Apache Tomcat
* Tools: Eclipse IDE, Git, GitHub

## 🔧 Modules Included

* User Authentication
* Laundry Booking System
* Admin Dashboard
* Order Management
* Customer Management
